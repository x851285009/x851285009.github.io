5. 栅格盒模型设计的精妙之处

容器两边具有15px的padding

目标 规则
列 两边具有15px的padding
行 两边具有-15px的margin
  • 为了维护槽宽的规则，
    • 列两边必须得要15px的padding
  • 为了能使列嵌套行
    • 行两边必须要有-15px的margin
  • 为了让容器可以包裹住行
    • 容器两边必须要有15px的padding
